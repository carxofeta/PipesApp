DecimalPipe y porcentajes

Pipes incluídos en Angular - Usualmente en Common Module.

{{ ventasNetas | number:'1.2-2' }} (En formato español)
{{ ventasNetas | number:'1.2-2':'en'}} (En formato anglosajón)
{{ ( ventasNetas + 2000000 ) | currency:'EUR':'symbol-narrow':'1.0-4' }} (en formato español)
{{ ( ventasNetas + 2000000 ) | currency:'EUR':'symbol-narrow':'1.0-4':'en'}} (En formato anglosajón)
{{ porcentaje | percent:'2.2-2' }} (en formato español)
{{ porcentaje | percent:'2.2-2':'en' }} (en formato anglosajón)
